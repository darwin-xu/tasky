# Tasky

A simple task tracking tool featuring an infinite canvas interface.

## Structure

- `backend` – Express.js API server
- `frontend` – React + Vite client

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend expects the backend to be running on `http://localhost:3001`.
